# Setup Discord Bot

1. Download this discord bot file
2. Open in Visual Studio Code
3. Go to Discord Developer
4. Go to section Bot
5. scroll down and clink on
   - Presence Intent
   - Server Members Intent
   - Message Content Intent
7. Add bot to your Discord Server witch All Permission
8. Go to back to Visual Studio Code and Click on Terminal and New terminal
9. Write to terminal python3 main.py
10. And have you good day :)
